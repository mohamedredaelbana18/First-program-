## 📝 ملاحظات الإصدار

### كيفية التحديث
1. قم بتشغيل `npm install` لتحديث التبعيات
2. قم بتشغيل `npm run db:push` لتحديث قاعدة البيانات
3. قم بتشغيل `npm run dev` لاختبار التطبيق

### التوافق
- Node.js 18+
